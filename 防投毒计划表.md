# 防投毒计划表

## 目标
在不影响现有检测流程的前提下，新增“订阅链接防投毒（基础版）”，默认策略为**警告继续**。

## 总体策略
- 版本范围：v1（基础版）
- 默认策略：检测到高风险先告警，不阻断检测
- 后续可切换：默认阻断 / 仅记录日志

## 任务拆分

| 编号 | 模块 | 任务 | 状态 | 优先级 | 验收标准 |
|---|---|---|---|---|---|
| AP-01 | 数据模型 | 新增风险等级、命中项、总报告结构体 | 待开始 | P0 | 可表达 Low/Medium/High/Critical + 命中原因 |
| AP-02 | URL 规则 | 协议校验（http/https）、私网/回环目标识别 | 待开始 | P0 | 可命中并返回风险原因 |
| AP-03 | 配置规则 | external-controller、secret、allow-lan、mode、DNS/TUN 规则 | 待开始 | P0 | 可按配置给出中高风险 |
| AP-04 | 节点规则 | skip-cert-verify 比例、私网 server、认证字段异常 | 待开始 | P1 | 输出节点层风险汇总 |
| AP-05 | 引擎整合 | 下载订阅后、节点检测前执行防投毒评估 | 待开始 | P0 | Start 阶段即可拿到风险报告 |
| AP-06 | UI 展示 | 风险等级徽标、命中数、明细展开区 | 待开始 | P0 | 主界面可视化展示且不挤压布局 |
| AP-07 | 策略行为 | 默认“警告继续”，保留阻断/仅记录扩展位 | 待开始 | P0 | 高风险场景默认继续并明显告警 |
| AP-08 | 单元测试 | URL/配置/节点规则命中测试 | 待开始 | P0 | 每条规则均有正反样例 |
| AP-09 | 回归测试 | 不影响现有解析、探测、评分与 UI | 待开始 | P0 | 现有功能无回退 |
| AP-10 | 文档更新 | README + 指标说明新增“防投毒”章节 | 待开始 | P1 | 用户可理解风险来源和限制 |

## 里程碑
- M1（核心）：AP-01 ~ AP-05 完成，可输出防投毒报告
- M2（可用）：AP-06 + AP-07 完成，用户可见可操作
- M3（可发布）：AP-08 ~ AP-10 完成，进入 v1.0beta 发布候选

## 非目标（本期不做）
- 签名校验、公钥体系
- 证书 pinning
- 外部威胁情报联动
- 主动抓包级 DNS 泄漏取证
