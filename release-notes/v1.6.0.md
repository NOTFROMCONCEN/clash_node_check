# Clash Node Checker v1.6.0

## Highlights

- Added bulk import pipeline for mainstream clients and local sources: URL, local file, and local directory auto scan.
- Added export presets for Clash / FlClash / Karing, including one-click export of available nodes from the GUI.
- Added import path picker buttons (file/folder) and expanded top menu actions for detection, view control, and help.
- Switched update check away from GitHub REST API limits: use `releases/latest` redirect parsing, with `releases.atom` fallback.
- Added 6-hour local update cache to reduce repeated network requests.

## Included assets

- Windows executable bundle (.zip)
- Standalone Windows executable (.exe)
- SHA256 checksum for the archive
